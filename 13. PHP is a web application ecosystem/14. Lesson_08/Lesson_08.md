# Урок 14. Лекция: Доставка приложений

## План урока

- обновлять приложение на сервере без простоев
- разделять роли DevOps, разработчик и администратор
- работать с CI/CD и использовать в вашей разработке
- автоматически доставлять приложение на сервер и развертывать его
- работать по чек-листу во время релиза
- работать со стратегиями развертывания и выбирать нужную вашему приложению
