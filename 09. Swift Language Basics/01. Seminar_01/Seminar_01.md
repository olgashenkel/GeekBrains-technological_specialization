### Задание 2.

Создайте константу `a`, которая будет равна `10`, затем создайте переменную `b`, которая будет равна `0`.

Напишите код, если `a` больше или равно `7`, то увеличить `b` на `7`, а если меньше, то необходимо уменьшить `b` на `7`.

***Результат выполнения Задания № 2:***
```
let a: Int = 10;
var b: Int = 0;

if a >= b {
  b += 7
  print(b)
} else if (a <= b) {
  b -= 7
  print(b)
} else {
  print("числа равны")
}
```


### Задание 3.

Используя переменные и константы из предыдущего задания, напишите код: если `a + b > 15` и `b >= 7`, то `b` нужно увеличить в два раза, в противном случае `b` нежно увеличить на 3.

***Результат выполнения Задания № 3:***
```
let a: Int = 10
var b: Int = 0

switch (a + b > 15, b >= 7) {
case (true, true):
    b *= 2
default:
    b += 3
}

print(b)


// ИЛИ

switch b {
case _ where a + b > 15 && b >= 7:
    b *= 2
default:
    b += 3
}
```


### Задание 4.

Используя переменные и константы из предыдущего задания, напишите код: если `a` находится в диапазоне `от 0 до 7`, то вывести в консоль результат сложения `a` и `b`, если `a` в диапазоне `от 8 до 20`, то вывести в консоль результат вычисления `a - b`, если `a` в диапазоне `от 21 до 50`, то вывести в консоль результат перемножения `a` и `b`. Во всех остальных случаях вывести в консоль `Нет походящих диапазонов`.

***Результат выполнения Задания № 4:***
```
let a: Int = 10
var b: Int = 0

switch a {
  case 0...7: print("a + b = \(a + b)")
  case 8...20: print("a - b = \(a - b)")
  case 21...50: print("a * b = \(a * b)")
  default: print("Нет походящих диапазонов")
}
```


### Задание 5.

Используя переменные и константы из предыдущего задания, напишите код: если `a > 10`, `b < 20` и `a + b > 15`, то `b` уменьшить на `10`. Если  `a - b < 20`, то вывести в консоль `b`. В противном случае увеличить `b` на `7`.

***Результат выполнения Задания № 5:***
```
let a: Int = 10
var b: Int = 0

switch (a, b) {
case _ where a > 10 && b < 20 && a + b > 15:
  b -= 10
  print("b - 10 = \(b)")
case _ where a - b < 20:
  print("b = \(b)")
default:
  b += 7
  print("b + 7 = \(b)")
}
```

### Задание 9.

### Задание 11.

Используя переменные и константы из предыдущего задания, напишите код: в диапазоне `от 1 до 10`, если `a + текущее число в диапазоне > b`, то увеличить `b` на `30`. В противном случае уменьшить `b` на `10`.


***Результат выполнения Задания № 11:***
```
var a: Int = 10
var b: Int = 0

for i in 1...10 {
  if a + i > b {
    b += 30
    print(b)
  } else {
    b -= 10
    print(b)
  }
}
```


### Задание 12.

Создайте переменную `k` равную `5`. В диапазоне `от 20 до 30` (не включая 30), если текущее число больше `25`, то `k` уменьшить на `4`. В противном случае `k` увеличить на `10`.


***Результат выполнения Задания № 12:***
```
var k: Int = 5

for i in 20..<30 {
  if i > 25 {
    k -= 4
    print("i = \(i); k - 4 = \(k)")
  } else {
    k += 10
    print("i = \(i); k + 10 = \(k)")
  }
}
```
